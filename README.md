## 快速说明

- 跨平台桌面客户端：Windows 10/11、macOS
- 批量下载 `m3u8` 并输出 `mp4`
- 支持任务进度、暂停/继续、失败任务导出
- 支持主题切换与中/英/日界面语言切换（默认中文）
